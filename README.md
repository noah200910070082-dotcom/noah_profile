# Noah Xu — Personal profile

An original bilingual profile for [noahxu.org](https://noahxu.org), built with Astro and plain CSS.

## Local development

```bash
npm install
npm run dev
```

## Production checks

```bash
npm run check
npm run build
```

The site is static and can be deployed to Vercel, Netlify, Cloudflare Pages, or GitHub Pages. Preserve existing MX/TXT records when connecting the Namecheap domain.

## GitHub profile references

Patterns borrowed from high-star GitHub profile resources:

- `abhisheknaiidu/awesome-github-profile-readme`: keep the intro short and make the first screen memorable.
- `alexandresanlim/Badges4-README.md-Profile`: use badges only when they clarify identity or contact.
- `tandpfun/skill-icons`: show skills visually in a compact row instead of writing a long list.

For this project, the website stays as an interactive one-page pixel profile. A separate GitHub profile README can later use the same content in a simpler format: short intro, skill icons, featured projects, competition notes, and contact links.
